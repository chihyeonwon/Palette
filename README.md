# Palette
팔레트 동아리 1분, 3분, 5분 크로키 용 스탑워치 앱
```
개발자 : Won Chi Hyeon
개발의도 : 1분 3분 5분 시간 설정 버튼을 클릭하면 스탑워치의 시간이 설정한 시간에 맞게 초기화되며
시간이 줄어들다가 0분 0초가 되면 설정한 벨소리의 알림을 울리게 한다.
```
